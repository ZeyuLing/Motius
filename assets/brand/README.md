# Motius Brand Assets

`Motius` comes from Motion + Motive + Motus: motion models with intent.

## Logo

- `motius-logo.png`: primary horizontal lockup.
- `motius-preview.png`: preview copy used in docs and review threads.

## Visual Idea

The mark combines a flowing `M` trajectory with a human-motion gesture. The
blue-to-cyan shape represents generated motion and model flow; the coral node
marks intent or control. The wordmark keeps the project name readable for
GitHub, documentation, model cards, and future product surfaces.

## Palette

| Token | Hex | Use |
|---|---|---|
| Graphite | `#111827` | wordmark and primary text |
| Cyan | `#15D5FF` | motion origin / energy |
| Cobalt | `#2563EB` | model/platform core |
| Coral | `#F9735B` | intent / control signal |
| White | `#FFFFFF` | primary logo background |
