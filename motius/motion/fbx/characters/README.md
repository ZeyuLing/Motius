# Built-In Mixamo-Compatible Characters

`atlas.fbx`, `nova.fbx`, and `gear.fbx` are original procedural character
assets generated by `tools/build_mixamo_characters.py`. They use the common
`mixamorig:*` body-bone naming convention so the same retargeting path works
for these built-ins and for a separately licensed Mixamo-compatible character.

These files do not contain Adobe Mixamo characters, meshes, textures, or
animations. The three generated character assets are released under
[CC0-1.0](https://creativecommons.org/publicdomain/zero/1.0/).
