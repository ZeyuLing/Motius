# MaskControl Attribution

This package is an independent Motius implementation of **MaskControl:
Spatio-Temporal Control for Masked Motion Synthesis** (ICCV 2025) by Ekkasit
Pinyoanuntapong et al.

- Paper: https://arxiv.org/abs/2410.10780
- Official code: https://github.com/exitudio/MaskControl

The architecture builds on MoMask and OpenAI CLIP. The released upstream
checkpoint archive contains an MIT notice for its MoMask-derived source, while
the current MaskControl README also asks users to follow the licenses of all
upstream dependencies and links to a CC BY-NC-ND license. Users of released
weights must review and comply with the terms published by the original
authors. Motius does not relicense third-party checkpoints.
