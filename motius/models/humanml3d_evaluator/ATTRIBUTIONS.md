# Attribution

The matching-network architecture and HumanML3D evaluation protocol are a
framework-native port of the evaluator released with
[TM2T](https://github.com/EricGuo5513/TM2T), which follows the HumanML3D
text-motion matching protocol introduced by Guo et al.

The implementation is self-contained and does not import an external reference
repository at runtime. Released upstream checkpoints retain their original
license and attribution requirements.
