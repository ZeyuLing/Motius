# Motius Leaderboard Case Assets

This dataset stores the compact joint trajectories used by the interactive
all-case explorers in the public Motius leaderboards. It is an asset companion
to [ZeyuLing/Motius](https://github.com/ZeyuLing/Motius), not a new training
dataset.

| Folder | Population | Contents |
| ------ | ---------: | -------- |
| `m2t-humanml3d/` | 4,400 | HumanML3D GT motion plus TM2T, MotionGPT, MotionGPT3, and VerMo captions |
| `t2m-humanml3d/` | 4,042 | HumanML3D selected captions and aligned outputs from every measured T2M variant |
| `babel-sequential/` | 1,295 | BABEL GT, FlowMDM, MotionStreamer, MotionLab, and PRISM long-form episodes |
| `temporal-condition/` | 4,012 per setting | Six Temporal Control settings with aligned GT, CondMDI, FlowMDM, KIMODO, MotionLab, and MotionCanvas outputs |

Trajectories are SMPL-22 joint positions quantized per case to unsigned 16-bit
values. Explorers retain every second or third source frame and linearly
interpolate during 20/30 fps playback. The source frame count, stride,
quantization minimum, and quantization scale are recorded in each leaderboard's
manifest, so every visualized point has a deterministic decode.

Assets are generated by:

```bash
python tools/build_m2t_case_explorer.py --help
python tools/build_motion_case_explorer.py --help
python tools/build_temporal_case_explorer.py --help
```

Metrics never consume these quantized browser assets. Evaluation uses the
original full-precision motions; this dataset is only for qualitative review.
