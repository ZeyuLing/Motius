MS = MotionStreamer-272 evaluator. uTMR = universal SMPL-22 joints66 TMR evaluator. The uTMR FID column keeps the legacy raw-embedding scale for cross-row compatibility; canonical L2-normalized FID is stored with new evaluation artifacts and is not mixed into this snapshot. MaskControl therefore leaves this legacy FID cell empty and reports normalized FID in its Model Card. Lower FID and MM are better; higher R-Precision is better. Diversity is a reference statistic and is not ranked.

Slide, Float, and Jitter use the shared SMPL-22 joint-level physical metric implementation. PoseQ is the MBench NRDF pose-quality score. Dynamic is compared with the GT reference rather than minimized.

Paper benchmark rows reproduce metrics reported by the original papers when the released checkpoint is not available in the shared SMPL-22 evaluation protocol. They are not used for generated-method ranking.

Calibration rows quantify legacy SMPL-to-HML263 conversion loss and are not ranked as generation methods. uTMR sanity is tracked by the GT row in the semantic table.
