# Monocular Motion Capture · 3DPW Test

Verified-only Motius leaderboard for 3DPW Test, EMDB-1, and EMDB-2.
Benchmark data remains under its original license and is never uploaded to this
Space. The public artifact contains only aggregate metrics, method provenance,
and separately licensed qualitative inputs.

Leaderboard publication is paused until a suitable licensed SMPL world-motion
benchmark is available. Completed method inference runs and diagnostic metrics
are documented only in their Model Cards. Paper numbers and partial smoke tests
are not valid submissions.

Camera-coordinate and global metrics are displayed in separate protocol views.
SOMA-77 methods participate in body-model-independent comparison only through
the audited 15-joint name mapping; they do not receive synthetic SMPL PVE rows.
